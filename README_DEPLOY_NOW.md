## 🆘 TROUBLESHOOTING

### Issue: Site not indexing
**Solution**: 
1. Check robots.txt isn't blocking
2. Submit sitemap in Search Console
3. Request indexing manually
4. Wait 1-2 weeks

### Issue: Schema not appearing
**Solution**:
1. Validate with Rich Results Test
2. Check for JSON syntax errors
3. Ensure schema is in `<head>` or `<body>`
4. Wait 1-2 weeks for Google to process

### Issue: Low traffic
**Solution**:
1. Be patient (3-6 months for results)
2. Keep publishing content (2-4 posts/month)
3. Build backlinks (10-20/month)
4. Check Search Console for issues

### Issue: High bounce rate
**Solution**:
1. Improve page load speed
2. Make content scannable (headings, bullets)
3. Add internal links
4. Improve mobile experience
