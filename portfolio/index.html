🏆 IN-SPACe Competition 2026

CAN-7USAT Ground Control Station

Real-time ground control station for GITAM's rocket targeting 1000m AGL in the IN-SPACe Model Rocketry Competition 2026. Decodes 46-byte binary telemetry from a Teensy 4.1 flight computer over 900 MHz XBee, runs Kalman filter sensor fusion, maintains a 6-state flight state machine, and broadcasts live data over WebSocket to a React dashboard.

"The challenge wasn't writing the code — it was writing code that works once, correctly, on launch day, with a rocket already on the pad."
<5ms end-to-end latency 0.5ms packet decode 0% packet loss (23/23)
Python FastAPI WebSockets Kalman Filter PostgreSQL React TypeScript

🤖 Autonomous Systems

AUVBrain — Autonomous Underwater Vehicle

Full autonomous control stack for an underwater vehicle. No internet, no margin for error. Runs a deterministic observe → decide → act agent loop with 4-DOF thruster control and a safety monitor enforcing hard limits on depth, battery, temperature, and obstacle proximity before any decision executes.

"Designing the fallback logic was harder than building the main path. What does the vehicle do when the LLM times out at 20m depth?"
0.84ms agent loop p95 Offline LLM + rules fallback
Python FastAPI ROS2 Docker asyncio llama.cpp

MedVision AI — Diabetic Ulcer XAI

Medical image classification + structured clinical data to estimate ulcer risk — with full explainability showing WHY. Grad-CAM heatmaps show which image regions the model attends to; SHAP values show how each clinical feature influenced the prediction. Full MLOps pipeline with MLflow, DVC, Prometheus, and Grafana.

"Black-box predictions are a liability in clinical settings. The explainability layer isn't a nice-to-have — it's the entire point."
